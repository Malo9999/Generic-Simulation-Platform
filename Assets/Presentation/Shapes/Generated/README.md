# Generated Shapes

This folder stores artifacts from `GSP/Generator/Shape Lab`.

Because some code-review systems reject binary assets, PNG sprite outputs are intentionally not committed.

To regenerate baseline sprites:
1. Open Unity Editor.
2. Open **GSP/Generator/Shape Lab**.
3. Click **Generate Default Neon Pack**.

This will bake the default shape set and populate:
- `Dots/`
- `Glows/`
- `Rings/`
- `Blobs/`
- `Strokes/`
- `Library/ShapeLibrary.asset`
